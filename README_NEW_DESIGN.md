## üöÄ Getting Started

### View the New Design
Open `index_new.html` in your browser to see the new stunning design.

The original portfolio is still available at `index.html`.

### Files
- `index_new.html` - New portfolio HTML
- `styles_new.css` - Complete CSS with all styling
- `script_new.js` - JavaScript for interactions and animations

### Replacing the Original
When ready to go live with the new design:
```bash
# Backup the original
mv index.html index_old.html
mv styles.css styles_old.css
mv script.js script_old.js

# Activate the new design
mv index_new.html index.html
mv styles_new.css styles.css
mv script_new.js script.js
```
